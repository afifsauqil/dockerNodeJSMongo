#   node-mongo-docker-compose
An example to Dockerising a nodeJS & mongoDB app using docker-compose
#### Requirement 
- docker
- docker-compose
  
## How to start 


Run docker compose
```
docker-compose up --build 
```
Test container using curl
```
curl localhost:3006
```
or using Postman with GET and POST method
